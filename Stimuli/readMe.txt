The are four stimulus sequences (A, B, C and D)
Taking as example sequence A, the three tones that make up the sequence are:

A1, A2 and A3

AC = A control: this is the matching sequence, made up of A1 + A2 + A3

AT1 and AT2 are the mismatching (Target 1 and 2) stimuli, whose formulas is

A1' + A2 + A3
A1'' + A2 + A3
